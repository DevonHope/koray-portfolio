Minimalistic React site for Koray Ozkan (therapist).

- Built with Vite + React
- Features: Header, Who I Help, Approach, Services, About, Book CTA (email + Psychology Today link)

To run locally:
1. cd koray-portfolio
2. npm install
3. npm run dev

Notes:
- The contact form opens the user's email client with a pre-filled message. Replace the placeholder email in `ContactModal.jsx` with the practitioner's email before publishing.
- Copy or customize styles in `src/index.css` to match branding.
